# 价格过渡计算器 - 团队共享部署指南

无需服务器，以下几种方式都可以让团队使用。

---

## 🎯 推荐方案（按易用性排序）

### 方案1：使用免费在线托管（最简单）⭐⭐⭐⭐⭐

**适合**：需要经常更新、多人协作的场景

**步骤**：

#### 使用 GitHub Pages（推荐）
1. 注册 GitHub 账号：https://github.com/signup
2. 创建新仓库，命名为 `price-calculator`
3. 上传 `web-calculator.html` 文件
4. 进入仓库设置，找到 "Pages" 选项
5. 选择 `main` 分支，点击 Save
6. 等待 1-2 分钟，会获得一个链接，类似：
   ```
   https://你的用户名.github.io/price-calculator/
   ```
7. 把链接分享给团队，大家打开就能用

#### 使用 Vercel（国内访问快）
1. 注册 Vercel 账号：https://vercel.com/signup
2. 下载并安装 Vercel CLI
3. 在 `web-calculator.html` 所在目录打开终端，运行：
   ```bash
   npx vercel
   ```
4. 按提示完成部署，会获得一个免费域名
5. 分享域名给团队

**优点**：
- 完全免费
- 访问速度快
- 随时更新文件即可同步
- 支持自定义域名（可选）

---

### 方案2：局域网共享（无公网访问需求）⭐⭐⭐⭐

**适合**：办公室内网使用，不需要外网访问

#### Windows 共享文件夹
1. 将 `web-calculator.html` 放在共享文件夹
2. 团队成员访问：
   ```
   \\你的电脑IP地址\共享文件夹名\web-calculator.html
   ```
   例如：`\\192.168.1.100\share\web-calculator.html`

#### macOS 共享
1. 系统设置 → 通用 → 共享 → 文件共享
2. 添加包含 HTML 文件的文件夹
3. 团队成员访问：
   ```
   afp://你的电脑IP地址
   ```

**优点**：
- 无需注册任何账号
- 速度快，局域网访问
- 完全免费

---

### 方案3：免费云盘分享（临时方案）⭐⭐⭐

**适合**：临时使用，团队规模小

**步骤**：

#### 使用百度网盘
1. 上传 `web-calculator.html` 到百度网盘
2. 生成分享链接
3. 分享给团队，下载后双击打开

#### 使用坚果云
1. 上传文件
2. 开启 Web 版预览功能
3. 团队可直接在浏览器中预览使用

#### 使用 WPS Office 在线文档
1. 将 HTML 内容复制到 WPS 在线文档
2. 插入 HTML 代码块
3. 团队协作编辑

**优点**：
- 简单快速
- 无需技术门槛

**缺点**：
- 每次更新需要重新上传
- 下载后使用，不够便捷

---

### 方案4：使用 VS Code Live Server（本地共享）⭐⭐⭐

**适合**：技术人员，需要本地调试

**步骤**：
1. 安装 VS Code
2. 安装 "Live Server" 扩展
3. 右键 `web-calculator.html` → "Open with Live Server"
4. 查看本地地址，例如：`http://127.0.0.1:5500/web-calculator.html`
5. 使用内网穿透工具（如 ngrok）分享给团队

```bash
# 安装 ngrok
brew install ngrok  # macOS
# 或者访问 https://ngrok.com 下载

# 启动内网穿透
ngrok http 5500
```

会获得一个公网地址，类似：`https://xxx.ngrok.io/web-calculator.html`

**优点**：
- 实时更新，修改立即生效
- 适合开发调试

---

### 方案5：使用免费静态托管平台（无需代码）⭐⭐⭐⭐

#### 使用 Netlify Drop
1. 访问 https://app.netlify.com/drop
2. 直接把包含 `web-calculator.html` 的文件夹拖进去
3. 等待上传完成，自动获得免费域名
4. 分享链接给团队

#### 使用 Cloudflare Pages
1. 注册 Cloudflare 账号
2. 创建 Pages 项目
3. 直接上传文件
4. 获得免费域名

**优点**：
- 全球 CDN 加速
- 支持自定义域名
- 完全免费

## 🎯 最终推荐

### 如果你想要最简单的方案：
→ **使用 GitHub Pages**
- 免费、稳定、访问快
- 全程有向导，5分钟搞定
- 链接永久有效

### 如果只是在办公室内使用：
→ **局域网共享文件夹**
- 无需任何注册
- 团队访问速度最快

### 如果你想要国内访问速度最快：
→ **Vercel 或 Netlify**
- 国内访问速度快
- 支持 HTTPS

---

## 📝 更新文件后的操作

无论使用哪种方案，更新文件后：

- **GitHub Pages**：更新仓库，自动部署（1-2分钟）
- **Vercel/Netlify**：重新上传文件，自动更新
- **局域网共享**：直接覆盖文件，立即生效
- **云盘分享**：重新上传并分享新链接

---

## ⚠️ 注意事项

1. **数据安全**：
   - 工具是纯前端，数据不会上传到任何服务器
   - 所有计算在浏览器本地完成

2. **浏览器兼容性**：
   - 建议使用 Chrome、Edge、Firefox、Safari 最新版
   - 不支持 IE 浏览器

3. **网络要求**：
   - 只需要一次下载，之后可离线使用
   - 无需保持网络连接
