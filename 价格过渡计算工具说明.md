# 价格体系过渡计算工具使用说明

## 📊 工具功能

**价格过渡计算器**（price-transition-calculator.html）用于帮助客户从体系一（首重+续重）过渡到体系二（公开价×折扣×新燃油），核心功能：

1. **反推折扣率**：输入客户旧体系价格，自动计算体系二下应该给多少折扣
2. **价差分析**：对比新旧体系价格，显示差异和波动幅度
3. **折扣建议**：自动生成建议折扣率，保持价格平稳过渡
4. **支持新老燃油费率分离**：
   - 体系一（老报价）燃油费率：可能是0%（报价已含燃油）或具体百分比
   - 体系二（新报价）燃油费率：跟随官网浮动（默认25%）

---

## 🚀 使用步骤

### 基础计算（0.5kg - 20kg）

1. 填写产品信息：
   - **产品名称**：例如"商业件标准"
   - **首重价格**：0.5kg的首重价格，如威海林道商业件标准首重50
   - **续重价格**：每增加0.5kg的续重价格，如续重13

2. 设置燃油附加费率：
   - **体系一（老报价）燃油附加费率**：
     - 如果老报价已经包含燃油费，填写 **0%**
     - 如果老报价需要单独加燃油，填写对应百分比
   - **体系二（新报价）燃油附加费率**：
     - 跟随官网浮动的燃油费率（默认25%）
     - 可根据官网当前费率调整

3. 点击 **"📊 计算折扣与价差"**

4. 查看结果：
   - 价格对比表显示所有重量段（0.5kg-20kg）的详细对比
   - 分析区域显示折扣率分析、价格波动分析、过渡建议

## 📋 结果解读

### 折扣率说明
- **反推折扣率**：精确计算值，基于"体系一含老燃油价格 = 公开价 × 折扣 × (1 + 新燃油费率)"反推
- **建议折扣率**：略低于平均值，可避免涨价情况

### 价格对比表
| 列 | 说明 |
|-----|------|
| 重量 | 0.5kg - 20kg，每0.5kg一档 |
| 公开价 | 体系二的基础公开价（不含燃油） |
| 体系一（不含燃油）| 老报价基础价 = 首重 + 续重 × 续重段数 |
| 体系一（含老燃油）| 老价格 × (1 + 老燃油费率) |
| 体系二（不含燃油）| 公开价 × 折扣 |
| 体系二（含新燃油）| 公开价 × 折扣 × (1 + 新燃油费率) |
| 反推折扣 | 根据老价格反推的折扣率 |
| 价差 | 新价(含新燃油) - 老价(含老燃油) |
| 波动 | 价差相对于老价格的百分比 |

### 价差颜色标识
- **绿色（diff-up）**：新价格更高（涨价）
- **红色（diff-down）**：新价格更低（降价）
- **灰色（diff-same）**：价格持平

## 🔄 核心计算逻辑

### 体系一（老报价）
```
不含燃油：价格 = 首重 + 续重 × 续重段数
含老燃油：价格 = 不含燃油价格 × (1 + 老燃油费率)
```

### 体系二（新报价）
```
折扣反推：
折扣 = 体系一含老燃油价格 ÷ [公开价 × (1 + 新燃油费率)]

含新燃油价格：
价格 = 公开价 × 折扣 × (1 + 新燃油费率)

不含燃油价格：
价格 = 公开价 × 折扣
```

### 价差计算
```
价差 = 体系二含新燃油价格 - 体系一含老燃油价格
波动百分比 = (价差 ÷ 体系一含老燃油价格) × 100%
```

## ⚠️ 注意事项

### 1. 老燃油费率的填写
- **关键**：仔细确认客户的老报价是否已经包含燃油费
- 如果老报价已含燃油，一定要填 **0%**
- 如果老报价需单独加燃油，填写对应百分比（如25%）
- 填写错误会导致折扣反推不准确

### 2. 新燃油费率的更新
- 新体系燃油费率跟随官网每月浮动
- 使用时请确认官网当前费率并更新
- 燃油费率变化会影响折扣反推结果和价格波动

### 3. 折扣率建议
- 建议折扣略低于平均折扣，可避免因燃油费率波动导致的涨价
- 不同重量段的折扣率可能略有差异，建议使用统一的折中值
- 折扣保留2-3位小数（如16.9%或16.88%），减少价格波动

### 4. 过渡时机
- 当新老燃油费率差异较大时，价格波动会更明显
- 建议在燃油费率相对稳定的时期进行过渡
- 如果波动>3%，建议调整折扣率或分批过渡

## 📌 使用场景示例

### 场景1：老报价已含燃油（老燃油费率=0%）
**客户情况**：
- 老报价：首重50元（已经包含燃油费，无需额外加）
- 体系二燃油费率：25%（跟随官网）

**设置**：
- 首重：50
- 续重：13
- 体系一燃油附加费率：**0%**
- 体系二燃油附加费率：**25%**

**结果**：
- 工具会自动反推折扣，确保含25%燃油后的总价与老报价一致
- 如果折扣为16.9%，新价格 = 990 × 16.9% × 1.25 ≈ 209（约等于老报价）

### 场景2：老报价需加燃油（老燃油费率=0%）
**客户情况**：
- 老报价：首重50元（不含燃油），需额外加0%燃油
- 体系二燃油费率：25%（跟随官网）

**设置**：
- 首重：50
- 续重：13
- 体系一燃油附加费率：**0%**
- 体系二燃油附加费率：**25%**

**结果**：
- 工具会基于"老报价×(1+0%)"反推折扣
- 确保过渡后含25%燃油的总价与老报价一致

### 场景3：燃油费率调整
**情况**：官网燃油费率从25%调整到30%

**操作**：
- 将"体系二燃油附加费率"从25%改为30%
- 点击"计算折扣与价差"
- 查看新燃油费率下的价格波动

**结果**：
- 系统会自动重新反推折扣
- 显示在新燃油费率下的价格波动情况
