## 业务规则 - 时效计算（重要，不得遗漏）

### ⚠️ 北海道偏远超区时效规则（必须遵守）
- 北海道是佐川**偏远超区**，派送时效 **3日以上**（在佐川标准时效基础上额外叠加）
- 计算时效时，必须同时考虑：
  1. 国内发货→大阪落地：约1天
  2. 清关+佐川提货：约1天
  3. **佐川周末不派送**：如落地或提货在周六/周日，顺延至周一
  4. 北海道偏远超区：佐川额外 **3日以上**
- 综合时效：明天（周六）发货 → 北海道预计 **5-7天** 送达
- 任何时候计算北海道时效，必须把偏远超区因素加进去，不能只看标准时效

### 佐川超区附加费
- 北海道：偏远超区，时效3日以上（在正常时效基础上叠加）
- 冲绳：+¥10/kg，最低¥150/票
- 离岛：单票单询

### 周末规则
- 佐川周六可提货，但**周末不派送**（顺延至周一）

## 合同法律风险要点（义达跨境国际货运代理服务协议）

> 文件来源：附件0《国际货运代理服务协议20260323》+ 附件1《告客户通知书》

### 乙方（义达跨境）核心免责条款
1. **货物包装责任全归甲方**：因包装不当导致损坏、延误、异形费等，全由甲方承担；包装不达标乙方有权拒收
2. **单证责任全归甲方**：托运资料不真实、不完整、低报/瞒报，导致扣关、罚款、无法清关，责任及风险甲方承担，仍需支付运费
3. **收货人问题不担责**：地址错误、拒收、无人收货等产生的仓储、二次派送、退货费，甲方承担
4. **收货后异议不受理**：外包装完好情况下收货人提出内装短少、收签后再提异议，乙方不承担任何责任
5. **货物转委托权**：乙方可将业务转委托第三方，无需单独取得甲方同意
6. **特殊货物须书面声明**：贵重品、易碎品、带电池电子产品、危险品等，须提前书面声明并得到乙方书面确认，否则按普货处理

### 甲方（客户）核心义务与风险
1. **禁止发仿牌/假货/违禁品**：
   - 库内查出：授权乙方直接销毁，承担销毁费用
   - 中国海关扣查：承担全部处罚 + 赔偿乙方 **¥2000/票**，仍需付运费
   - 境外海关扣查：承担全部处罚 + 赔偿乙方 **¥10000/票**，仍需付运费
2. **普货/带电货严格区分，不得冲货**
3. **舱位确认后不得退关**：临时取消产生的亏/空舱费甲方承担
4. **3个工作日内须回复**：乙方协商解决方案后，甲方须3工作日内答复，否则损失自担
5. **迟延付款违约金**：每逾期1日，按欠款金额万分之五支付违约金；乙方可留置货物直至结清

### 赔偿上限（乙方责任期内）
- **无投保**：最高赔偿 **$100/票**（取申报货值、采购货值、$100三者最低值）
- **有投保**：按保险理赔流程
- **不承担连带损失**（高价值货物建议客户自购保险）
- 收到赔付通知后**7天内**须提供采购发票等资料，否则乙方可暂缓赔付

### 争议解决
- 适用**中华人民共和国法律**（不含港澳台）
- 协商不成：向**乙方注册地（上海市青浦区）有管辖权的人民法院**提起诉讼
- 维权费用（律师费、诉讼费等）由违约方承担

### 合同有效期
- **3年**，自双方签署之日起生效
- 到期未续签但继续开展业务，视为继续沿用本协议
- 任何一方提前**30天书面通知**可解除协议
