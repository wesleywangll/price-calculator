# 2026-04-01 工作记录

## 价格过渡计算工具升级

更新价格过渡计算工具（price-transition-calculator.html），支持分别输入新老体系的燃油附加费率：

### 更新内容
1. **燃油附加费率分离**：
   - 体系一（老报价）燃油附加费率：独立输入，可能是0%（报价已含燃油）或具体百分比
   - 体系二（新报价）燃油附加费率：独立输入，跟随官网浮动（默认25%）

2. **计算逻辑调整**：
   - 折扣反推基于：体系一含老燃油价格 = 公开价 × 折扣 × (1 + 新燃油费率)
   - 价差计算对比：新价格（含新燃油）vs 老价格（含老燃油）
   - 分析结果显示新老燃油费率对比

3. **界面优化**：
   - 将燃油附加费率输入分为两个独立字段
   - 添加说明文字：老燃油费率如果报价已含燃油则填0%
   - 更新表头：显示"体系一（含老燃油）"和"体系二（含新燃油）"
   - 修复重量显示顺序：使用 Object.keys().sort() 确保 0.5→20 递增显示

4. **文档更新**：
   - 更新价格过渡计算工具说明.md
   - 添加三个使用场景示例
   - 强调老燃油费率填写的注意事项

## Web端版本创建

创建 web-calculator.html，适合部署到服务器供团队使用：

### 主要功能
1. **响应式布局**：支持桌面、平板、手机自适应
2. **左侧输入 + 右侧结果**：更符合用户操作习惯
3. **导出Excel**：一键导出CSV格式，支持Excel打开
4. **打印优化**：专门的打印样式，方便纸质归档
5. **使用说明**：左侧面板内置核心原则和关键提示
6. **现代化UI**：渐变背景、卡片式设计、悬停效果

### 使用方式
- 直接打开HTML文件即可使用
- 可部署到任意Web服务器（Nginx/Apache/静态托管）
- 无需后端，纯前端实现
- 支持Chrome/Edge/Firefox/Safari等现代浏览器
